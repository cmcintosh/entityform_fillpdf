####
# Description
####
This module creates a bridge between the EntityForm module and FillPDF, allowing you to create
PDF forms that have been filled out with the values from a EntityForm Submission.

####
# Installation
####

Follow the installation instructions for EntityForm and FillPDF.  Ensure that you apply the patch #8 and #10 from
https://www.drupal.org/node/1526452 onto FillPDF to allow token support from entities.
